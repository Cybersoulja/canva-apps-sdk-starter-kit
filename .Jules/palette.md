## 2025-02-17 - External Link Button UX
**Learning:** For buttons that open external links, it's a good practice to indicate that the user will be taken to a new tab/window by adding an external link icon.
**Action:** When using `@canva/app-ui-kit` to create an external link button, use `icon={OpenInNewIcon}` and `iconPosition="end"` to improve the user experience.
