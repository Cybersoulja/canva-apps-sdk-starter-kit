## 2026-04-19 - External Link Visual Indicator
**Learning:** Adding `OpenInNewIcon` to external links using the Canva app UI kit button component (`icon={OpenInNewIcon} iconPosition="end"`) improves accessibility by setting clear expectations that an external tab will open, reducing context-switching surprises.
**Action:** Always append the `OpenInNewIcon` to any external navigation buttons using this pattern from the `@canva/app-ui-kit`.
